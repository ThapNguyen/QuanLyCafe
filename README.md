### Các chức năng chính
1. Quản lý nguyên vật liệu
	- Thêm nguyên -vật liệu 
	- Sửa nguyên -vật liệu
	- Xóa nguyên -vật liệu
	- Tìm kiếm nguyên -vật liệu
	- Liệt kê kho hiện tại
2. Quản lý Menu
	- Thêm thông tin Menu
	- Sửa thông tin Menu
	- Xóa thông tin Menu
	- Tìm kiếm thông tin Menu
3. Quản lý hóa đơn 
	- Thêm hóa đơn
	- Sửa hóa đơn.
	- Xóa hóa đơn. 
	- Liệt kê các hóa đơn
4. Báo cáo thống kê
	- Báo cáo doanh thu theo mốc thời gian
	- Thống kê mặt hàng bán chạy nhất/bán ít nhất
	- Thống kê lợi nhuận
